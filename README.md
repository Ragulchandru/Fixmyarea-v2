# FixMyArea 🚧

FixMyArea is a community-driven web platform that enables citizens to report local civic issues such as potholes, garbage accumulation, streetlight failures, water leakage, and other public infrastructure problems.

## Features

- 📸 Upload issue photos
- 📍 Automatic GPS location detection
- 🤖 AI-generated complaint descriptions
- 🗺️ Interactive map-based issue reporting
- 📊 Public dashboard for transparency
- 🔄 Real-time status tracking
- 👨‍💼 Admin management system
- 📱 Responsive design for mobile and desktop

## Tech Stack

## Installation

### Clone Repository

```bash
git clone https://github.com/Ragulchandru/Fixmyarea-v2.git
cd Fixmyarea-v2
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```
